# YouTube Channel Data Analysis 🚀

## Workign with a Kaggle Dataset on YouTube Channel Analytics! 📝

## Tools used 🔨
- Python
- Pandas
- SQLite3

## Installation
1. cd Tracker-Real
2. Install virtual environment: python -m venv venv
3. Activate the virtual environment A. On MacOS/Linx: source venv/Scripts/Activate, B. For Windows: venv/source/activate
4. Install requirements: pip install -r requirements.txt
5. Run application: python main.py
